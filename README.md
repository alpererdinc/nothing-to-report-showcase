# Nothing To Report

**Nothing To Report** is a 2D cyber-noir newsroom game prototype where the player works behind the desk of an independent media channel in a corrupted future city.

The city says there is nothing to report.  
The documents strongly disagree.

## Overview

In **Nothing To Report**, the player investigates official reports, field notes, camera logs, voice messages, and backchannel packets to uncover contradictions behind each case.

The game is built around interface-driven gameplay. Instead of controlling a character directly most of the time, the player works through documents, evidence, messages, reports, and risky editorial decisions.

The main question is not only:

> What is true?

but also:

> How much of the truth can we publish without getting someone hurt?

## Gameplay

The current playable demo includes:

- Document investigation
- Contradiction detection
- Headline decision system
- Consequence-based stats
- Visual evidence inspection
- Field reporter guidance minigame
- Backchannel signal routing minigame
- Retro terminal UI and CRT-inspired presentation

## Core Stats

Player decisions affect several newsroom and story variables:

- **Public Trust**
- **Danger**
- **Money**
- **Reporter Safety**

Publishing the truth can increase public trust, but it may also raise danger.  
Playing it safe can protect the team, but it may slowly turn the newsroom into furniture.

## Characters

### Mara

Mara is the editor guiding the player through the night shift.  
She is experienced, tired, sarcastic, and usually one bad official statement away from losing patience.

### Nox

Nox is a field reporter who keeps sending unstable messages from places he probably should not have entered.

He is useful.  
This is not always the same thing as safe.

## Current Demo

The current demo is designed as a short playable slice of the game. It introduces the main gameplay loop and several planned systems:

1. Read documents.
2. Find contradictions.
3. Make editorial decisions.
4. Protect field reporters when possible.
5. Route unstable information through unofficial channels.
6. Try not to become official media by accident.
